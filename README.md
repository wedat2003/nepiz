# Nepiz

Private relationship website built with Next.js.

## Main Pages

- Home
- Gallery
- Our Journey
- Secret
- Counter
- Date Idea Roulette
- Wish List
- Map of Us
- Open When

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Leaflet

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- This repository contains the current active version.
- Data is stored in browser storage and cloud sync endpoints configured in the app.
